# Pemrograman Web

Mata Kuliah Pemrograman Web mengenalkan pemrograman web, termasuk sejarah dan bahasa-bahasa pemrograman web populer, kursus ini meliputi topik-topik seperti HTML, CSS, dan JavaScript untuk membangun tampilan web, hingga penggunaan Node.js dan MySQL untuk mengembangkan aplikasi web yang dinamis. Selain itu, Mahasiswa juga akan mempelajari konsep-konsep keamanan web, teknik deployment aplikasi web, dan akan memiliki kesempatan untuk menerapkan semua yang telah dipelajari dalam sebuah proyek akhir. Dengan demikian, kursus ini bertujuan untuk mempersiapkan Mahasiswa dengan keterampilan dan pengetahuan yang diperlukan untuk menjadi seorang pengembang web yang kompeten dan dapat menghasilkan aplikasi web yang berkualitas.
